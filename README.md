# HolyRedeemer

School library management system with RFID/QR circulation, fine tracking, gamification, analytics, and PostgreSQL.

---

## Features

- **RFID/QR Circulation**: Quick book check-in and check-out using RFID tags or QR codes
- **Fine Tracking**: Automated overdue fine calculation and management
- **Gamification**: Reading challenges, badges, and leaderboards to encourage reading
- **Analytics**: Dashboard with circulation stats, popular titles, and usage trends
- **PostgreSQL**: Robust relational database for reliable data storage

---

## Tech Stack

- **Database**: PostgreSQL
